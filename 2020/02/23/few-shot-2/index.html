CROSS-DOMAIN FEW-SHOT CLASSIFICATION VIA LEARNED FEATURE-WISE TRANSFORMATION

https://openreview.net/pdf?id=SJl5Np4tPr

总结

提出了一个feature-wise变换层的结构，并提出一个learning-to-learn算法，通过使用多个已知域模拟泛化过程来优化feature-wise变换层的超参。实验表明该方法可以用于多个基于度量的few-shot分类算法，并显示出优于baseline的性能。

Problem definition

主要目标为改进在训练与测试图片域不同时造成的性能下降问题，如下图所示

Task $T$包括Support set$S={(X_s,Y_s)}$和Query set$Q={(X_q,Y_q)}$

特征提取器$E$首先提取出support和query中的特征，之后用度量函数$M$通过support图片的标签$Y_s$、编码后的support和query图片$E(x^s)$和$E(x^q)$来预测query图片的类别，如式(1)所示。

最后，训练的目标就是query set中的分类loss，即式(2)。

在本论文中，定义一个域中包含一个few-shot分类任务的集合，即$T={T_1,T_2,…,T_n}$。假设在训练阶段有$N$个已知的域${T_1^{seen},T_2^{seen},…,T_n^{seen}}$，目标是通过已知的域训练出一个模型能对于未知的域$T^{unseen}$有较好的效果。
需要注意的是在训练阶段模型不接触到未知域$T^{unseen}$中的图片数据内容。

Approach

FEATURE-WISE TRANSFORMATION LAYER

由于域间差异，度量函数$M$很可能会对已知域过拟合，因而对未知域的拟合效果变差。为了解决这个问题，在特征提取器中$E$加入包含仿射变换的feature-wise变换用于增加中间特征的激活性。直观地说，有了feature-wise变换层特征提取器$E$能够产生更为分散的特征分布，因而度量函数$M$的泛化能力得到增强。变化层结构如图2(a)所示。

LEARNING THE FEATURE-WISE TRANSFORMATION LAYERS

虽然超参$\theta_f={\theta_\gamma,\theta_\beta}$可以凭经验设置，但事实上手动调整一组对于不同设定有着不同效果参数还是较难的。为了解决这个问题，文中设计出了一种learning-to-learn算法来调整feature-wise变换层的超参。

核心思想是使用已知域训练基于度量的模型应该能够使模型在未知域中的表现有提升。算法如下图所示

在每一次训练迭代$t$中，从已知域${T_1^{seen},T_2^{seen},…,T_n^{seen}}$中采样获得一个pseudo-seen(伪知的)域$T^{ps}$和一个pseudo-unseen(伪未知的)域$T^{pu}$。设模型特征提取器为$E_{\theta_e^t}$，度量函数为$M_{\theta_m^t}$，首先使用超参数$\theta_f={\theta_\gamma,\theta_\beta}$整合进特征提取器中，然后使用式(2)中的loss通过pseudo-seen task$T^{ps}={(X_s^{ps},Y_s^{ps}),(X_q^{ps},Y_q^{ps})}$更新模型的参数，即下式(5)。

Experimental results

表1中FT表示使用包含由经验设定超参的feature-wise变换层来训练模型。由表可得，在5-way 1-shot的情况下使用FT对于未知域中的分类结果大多数有1~2%的提升，在5-way 5-shot的情况下多数有1~4%的提升。通过加入FT，GNN模型显现出了STOA的性能。
